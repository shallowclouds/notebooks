#### shadowsocks-libev

```bash
$ sudo apt-get install software-properties-common	 	 
$ sudo add-apt-repository ppa:max-c-lv/shadowsocks-libev
$ sudo apt-get update
$ sudo apt-get install shadowsocks-libev
$ sudo systemctl start/stop/restart shadowsocks-libev.service
```
